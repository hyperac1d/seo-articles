Tools that helps you test your Site's Mobile Responsive Design


The first tool that I used to test my site's responsiveness is the google's mobile-friendly checker. This could be one of the best website that you can use because it is a google made site which is really a known company.

As you can see in the picture, it shows that my site is a mobile-friendly and this page is easy to use on a mobile device. BUT it also shows that it has a page loading issues. It is because that my pictures are too heavy and I didn't compress it yet. So there's a need to update my site and compressed all my images to lessen the size of the pictures.



The second site that I've use to test my site's mobile responsiveness is Website Grader. Test yours too!



It says that my site is good and show results for performance, mobile, seo and security.

It has a score of 23/30 because it has a good page size, which is light, it is about 72KB and a page request is 8 and the page speed is high, about 6.2 seconds. This could be sucks as users will leave your site as it loads so slow. Therefore, there is a need for my images to compressed.

It has a mobile responsive score of 30/30 it is because I set in the head tag the viewport of the page which will be resize depending on the device width. This should actually high because of the time, people usually used mobile devices anywhere.

The third part of scoring is the SEO Which is very low. So the problem of my blog is that I didn't add meta description tags and a site map for users to know where they are in your blog.



So the third site I used was Responsive Design Checker


This site is to actually see your site from different devices such as desktop in different sizes of monitor, different types of tablets, and different types of phone such as iphone5s, sony xperia x2, etc. Rather than showing results of the mobile friendliness, you will be the one actually see it and test it.
