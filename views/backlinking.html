Importance of URL Canonicalization

person Matildo, M.
date_range July 26, 2017

URL Canonicalization is so important as it typically helps you choose the best URL, typically for home pages. You can set an URL on how search engine shows your site. You can also set up key redirects that will allow users flexibility in the URL they input while providing redirection to a canonical URL and delivering consistency for search engine results.


HTTP Status Codes

HTTP Status Codes are 3 digits long example: 1xx/2xx/3xx/4xx/5xx.

1xx: Information - Code starting in 1 gives information while the connection is still in progress.

Example
  1. 100: Continue
  2. 101: Switching Protocol
  3. 103: Checkpoint

2xx: Successful - means that successful connection was made. 200 means OK, while 201 to 206 means that there's more happening like resources recreated.

3xx: Redirection - indicates redirection like 301: permanent redirects

4xx: Client Side Error - indicates that there's a client side error. Example code is 403, which users are not allowed to access or forbidden or 404 if the page was not found or does not exist.

5xx: Server Side Error indicates that there is a server side error. Like failed database connection or the server is being down.
